주제 정하기

1. 시장 데이터 분석

주제 : 주가 시장 데이터와 다른 요인들을 분석하는 것으로 상승세와 하락세 예측/보편화

데이터 : 주가 시장 데이터, 다른 각종 요인들

회귀분석

종목, 회사를 정해서 데이터를 가져온다. 데이터 출처는 구글 또는 네이버

2. 유튜브 쇼츠 추천재생목록 제공 모델

주제 : 데이터 분석을 통해서 추천목록을 만드는게 가능한가? 그리고 만들어지면 어떤 추천목록이 나오는가, 어떤 종류의 영상이 잘 먹힐것인가

데이터 : 유튜브 API - 영상 시청한 기록, 좋아요 개수, 시청한 영상을 본 사람이 본 또 다른 영상, ...

3. 부동산 데이터 분석(집값 예측모델, 지도시각화 연습)
참고 : https://dacon.io/competitions/official/235724/overview/description

주제 : 데이터 분석을 통해 집값 예측이 가능한가(ex. 서울시)

데이터 : 집값, 지역에서 편의점 분포, 편의시설 정보

수치화가 가능할까?, 상승할것이다, 하락할것이다를 예측?, 시각화

4. 사회문제, 사회가치 해결 및 증진 - 사회문제에서 나오는 데이터를 분석 후 행결방안 제시

5. 대선테마주를 선거전, 1년전, 이평선 120선 밑에 있었을때는 기준으로 대선때 까지의 주가흐름

6. 지역별, 성별, 연령 등으로 정리할 수 있는 코로나 감염자 현황

결과 : 3. 부동산 데이터 분석

데이터 참고 주소
	- https://dacon.io/competitions/official/235724/overview/description
	
데이터 탐색 범위 : 구 단위 (서울시)
	시 단위가 아닌 구 단위를 선택한 이유 : 구 단위로도 편차가 심할 것 같았기 때문에 시 단위로는 데이터의 가치가 낮을 거라고 생각했음

부동산 가격을 예측하는 지표
	단기 : 주택거래량 / 전세 값 / 미분양 물량 / 인허가 물량 / 주택담보대출잔고추이
	
	3~5년 : 인구구조 / 소득구조 / 정책기조 / 금리기조 / 수급변동 / 글로벌 부동산 동향 / 환율

데이터 탐색 목적
	- 서울~경기권에 내가 살 집
		전월세 지도와 부동산 가격 지도 형식으로 지도화
		데이터 예시 : 편의시설(학교, 대중교통), 경제활동 단지의 여부, 특화도시 여부, 도로/시설 연결지역,
				유동인구 및 유동인구의 원인, 지역별 순인구수, 교통문화지수, 교육자 1인당 학생수, 의료 및 문화시설,
				정부의 부동산 정책, 수도권 인구집중화(세부내용 설정 필요), 주택거래량, 전세 값, 미분양 물량,
				인허가 물량, 인구구조, 수득구조, 정책기조, 수급변동, 글로벌 부동산 동향, 환율,
				지역별 순인구수, 교통문화지수, 교육자 1인당 학생수, 의료, 문화시설, 정부의 부동산 정책
		지도화 구역 : 서울시 구 단위
		시간대 : 2005.01.01 ~ 2019.12.31 / 2020.01.01 ~ 2021.10.31 - 총 약 16~7년
		데이터(종속변수) : 부동산 가격(매매가 / 전월세)
		요인(독립변수) : 도시별 인구밀도(인구수÷세대수), 도로연결현황(도로교통부),
